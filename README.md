Flask practice with kaggle titanic tutorial
===========================================

Commands
--------
```
make [command]
```

Example
-------
Build docker images for frontend and backend, then run both containers.

```
  make build
  make run
```
Access to the web application at localhost:8080 .

Clean up the web apps.

```
make clean
```


Install app
-----------
```
make install
```
